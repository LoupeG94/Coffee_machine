Lucky 7 Open on Hyperskill

Find all numbers that can be divided by 7 and print their squares.

The input format:

In the first line, there's an integer number of values n.

Then n lines with one number in each follow.

The output format:

m lines with the numbers that are divided by 7 squared.

[HINT]You can write n to some variable and then call input() function exactly n times, using for cycle.[/HINT]


Sample Input:
5
8
11
-49
0
3564
Sample Output:
2401
0


Memory limit: 256 Mb
Time limit: 5s
